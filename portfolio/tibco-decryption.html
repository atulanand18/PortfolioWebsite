Decrypting a password in TIBCO. Not a recommended practice. Only for exceptional scenario.

  • April 29, 2020
  • By Atul Anand
  • 1 minute read.


In this blog, I will talk about decrypting a password. Though this is not a recommended practice due to security concerns, but in case anyone needs it in an urgent exceptional scenario.

TIBCO provides ObfuscateEngine class within tibcrypt.jar which can be used to decrypt the password. The tibcrypt jar can be found at TIBCO_HOME/lib folder. You can create a java code and call ObfuscateEngine.decrypt() method by passing the encrypted password as an input to the method. The output returned by the execution of the method is the decrypted password.
