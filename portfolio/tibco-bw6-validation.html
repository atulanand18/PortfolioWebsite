Tibco Business Works 6 Project Validation.

  • July 31, 2019
  • By Atul Anand
  • 4 minute read.


In this blog, we will talk about TIBCO BW 6 project validation.

I came across a problem in my project where an unvalidated code somehow made it to the Dev server which resulted in application errors. The build process doesn't report any tibco project validation issues as it does for a normal java application project.

This problem statement led me to look for some solution so that any validation errors could be caught up during the build process itself. I looked for TIBCO documentation and went through multiple blogs but could not find a solution for it. However Tibco provides bwdesign system:validate function but that command can be executed as standalone only.

This is when I thought of leveraging Tibco's provided bwdesign system:validate function and created a simple Java application using it, which would run validation on TIBCO projects. I made it as a Java plugin : bw6-validation-maven-plugin which can be plugged-in any of the TIBCO BW 6 application projects.

bw6-validation-maven-plugin is a maven plugin for performing validation of TIBCO BW 6.x version projects during build process. It makes use of bwdesign utility for the execution part. It first creates a workspace and imports the application modules in the workspace using bwdesign's system:import. Finally it calls system:validate to perform validation of the workspace and report any validation errors.

The main benefit we get from this is that it reports any validation errors in the project which might have creeped in unintentionally and can be caught during the project build process.

To use bw6-validation-maven-plugin is simple. You download the code provided in the github location in the link shared at the end of the blog. Run mvn clean install which will create the plugin in your local git repository. Now include this plugin in your TIBCO application at the parent level and this would report any validation errors found in the project at build time.

<plugin>
   <groupId>com.tibco.bw6.plugins</groupId>
   <artifactId>bw6-validation-maven-plugin</artifactId>
   <version>1.0.0</version>
   <executions>
      <execution>
         <goals>
            <goal>ProjectValidate</goal>
         </goals>
      </execution>
   </executions>
</plugin>


The complete source code for the validation plugin is available at this GitHub project.
